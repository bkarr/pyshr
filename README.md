# pyshr
Python package for calling libshr functions

## Dependencies
[Libshr](https://github.com/bkarr/libshr) 0.17.0 or later

## Installation                                                                         
    ~ $ git clone https://github.com/bkarr/pyshr.git                            
    ~ $ cd pyshr                                                                
    ~/pyshr $ python setup.py install
